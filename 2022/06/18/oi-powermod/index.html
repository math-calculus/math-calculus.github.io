OI - 乘方取模算法

发表于2022-06-18|更新于2022-11-08|OI
|字数总计:806|阅读时长:3分钟|阅读量:

[ 提示：这是一个黑历史！ ]

给三个整数 a,b,c，求 a^b mod c的值.

要解决这个问题，首先可能想到的方法是(int)pow(a,b)%c.但是若数据太大，很有可能会溢出.

把问题简化一下，简化成 (ab) mod c.

要求这个算式有什么简便的方式？请读者先思考一下.

这里给出一种方式:(ab) mod c = [(a mod c)(b mod c)] mod c.

将 a 取模 b 后，a 被减去了 c 的整数倍，这些 c 的整数倍被 c 取模后一定为 0。
同理，b 被减去了 c 的整数倍，这些 c 的整数倍被 c 取模后一定为 0。

设a被减去了c的k1倍，b被减去了c的k2倍，把 (a-k1*c)(b-k2*c) 展开，得:

(a-k1*c)(b-k2*c) = a(b-k2*c)-(k1*c)(b-k2*c)
因为(k1*c) 被c取模后一定为0，所以 (k1*c)(b-k2*c)被c取模也一定为0，所以对最后对c取模没有影响，这里去掉，得：

a(b-k2*c) = a*b-a(k2*c)

因为(k2*c) 被c取模一定为0，所以 a(k2*c)被c取模也一定为0，所以对最后对c取模没有影响，这里也去掉，得：

a*b

把(a-k1*c)(b-k2*c)去掉一些对 c 取模没有影响的数后，最后得到:(a-k1*c)(b-k2*c)对c取模等于a*b对c取模，这就得到了之前的公式：

(ab) mod c = [(a mod c)(b mod c)] mod c

我们把问题放回原来的问题：给三个整数 a,b,c，求 a^b mod c的值.

首先，大家都知道，a^b等于b个a相乘，也就是a·a·a…·a.

那么我们可以把问题套在刚才那个简化版的问题上：求(ab) mod c.这里需要对a\mod c不断乘b-1次（因为a\mod c本身是一次），每乘了一次后获得的乘积，再对c取一次模。也就是(((a \mod c)(a \mod c))\mod c)(a \mod c))\mod c…在C++中很容易用循环实现这种操作。
